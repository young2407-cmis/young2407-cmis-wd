Procedure to Create and View on the Production Server

  1. Open Terminal
  2. Clone your repository
  3. Have terminal run in your repository with the cd command
  4. Open production server
  5. Create a new html file with gedit command
  6. insert doctype (html)
  7. Insert the selectors and elements such as header, body, main, nav, and footer
  8. Insert text in the right column of the selector
  9. Save it in your repository
  10. type localhost:8000/filename.html on the browser to view from the development server
  11. Qualify your product
  12. Use sftp command to log into sftp
  13. Upload the file using the command put
  14. go to http://cmis-hsteacher/~wooyoung2407/index.html to view from the production server
